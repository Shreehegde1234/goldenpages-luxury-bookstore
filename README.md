# GoldenPages Luxury Bookstore 📚✨

A premium, full-stack E-commerce Bookstore application built with the MERN (MongoDB, Express, React, Node.js) stack. This project features a sophisticated, "Midnight and Platinum" luxury aesthetic with high-end typography and smooth animations.

## 🌟 Key Features

- **Full Authentication**: Secure Login and SignUp using JWT tokens.
- **Luxury UI/UX**: Premium black, gold, and white color palette with modern, responsive design.
- **Smart Search**: Functional search for books with integrated search history.
- **Wishlist & Cart**: Seamlessly manage your favorite books and shopping basket.
- **Admin Dashboard**: Comprehensive dashboard for managing inventory and orders.
- **Secure Payments**: Professionalized payment processing integration.
- **User Experience**: Smooth micro-animations, mobile-responsive layout, and intuitive navigation.

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS, Vite
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **State Management**: React Context API
- **Auth**: JSON Web Tokens (JWT)

## 🚀 Getting Started

### 1. Backend Setup
```bash
cd backend
npm install
# Create a .env file with your MONGO_URI, JWT_SECRET, etc.
npm run dev
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
